Este projeto corresponde à Fase 2 da UC de Estruturas de Dados Avançadas (EDA), focando-\/se na representação e manipulação de grafos em linguagem C.

Cada antena é representada como um vértice de um grafo, e as adjacências entre antenas com a mesma frequência são representadas como arestas. ~\newline
 Foi implementado suporte a\+:
\begin{DoxyItemize}
\item Percursos em profundidade (DFS)
\item Armazenamento e leitura em ficheiros binários
\item Criação dinâmica de adjacências entre vértices
